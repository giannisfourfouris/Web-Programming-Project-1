Πρωτότυπα

Σε αυτόν τον ιστιότοπο θα προσπαθήσω να δημιουργήσω μια συλλογή σελίδων οι οποίες αφορούν και περιγράφουν εμένα. Σκέφτομαι να υιοθετήσω την δομή CSS grid αλλά με κάποιες τροποποιήσεις. Η δομή της σελίδας θα είναι η εξής:
  1. πάνω πάνω το header μαζί με το menu (δεξιά)
  2. στη μέση main μαζί με menu (Δεξιά) και
  3. κάτω κάτω footer μαζί με main (δεξιά).
Επίσης σκέφτομαι να χρησιμοποιήσω 4 σελίδες οι οποίες θα είναι οι εξής:
  1. Σχετικά με μένα (Σελίδα η οποία λέει γενικά πράγματα για μένα)
  2. Ακαδημαϊκές εργασίες (Σελίδα η οποία περιγράφει τις πιο ενδιαφέρουσες εργασίες στο πλαίσιο της σχολής με links που δείχνουν στον πηγαίο κώδικα)
  3. Ενδιαφέροντα (Σελίδα η οποία περιγράφει τα ενδιαφέροντα/Hobbies μου) και
  4. Επικοινωνία (Σελίδα η οποία περιέχει μία φόρμα επικοινωνίας αν θέλει κάποιος να έρθει σε επαφή μαζί μου).
